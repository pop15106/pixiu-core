# 深度思考引擎 — 數學推理範例

## 情境：使用者問一道邏輯題

**使用者**：一個房間裡有 100 個人，99% 是男性。要讓男性比例降到 98%，需要多少男性離開？

**模範回應**：

**推理過程：**

1. **初始狀態**：100 人，99% 男性 → 99 男 + 1 女
2. **目標狀態**：男性比例 = 98%，女性不變 = 1 人
3. **設 x 為離開後剩餘男性人數**：
   - 總人數 = x + 1（剩餘男性 + 1 女性）
   - 男性比例 = x / (x + 1) = 0.98
4. **解方程**：
   - x = 0.98(x + 1)
   - x = 0.98x + 0.98
   - 0.02x = 0.98
   - x = 49
5. **驗證**：49 男 + 1 女 = 50 人，49/50 = 98% ✅
6. **需離開人數**：99 - 49 = **50 人**

**結論**：需要 50 名男性離開。

**信心**：L5（數學推導，可驗證）

**舉一反三**：
- 這題的反直覺之處在於：比例只降了 1%（99%→98%），但需要離開一半的人。這是「百分比基數效應」——當分母很小時，微小的比例變化需要很大的絕對值調整。
- 同樣的思維可以用在：稀釋濃度問題、投票比例調整、品質指標改善成本估算。
